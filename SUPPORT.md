# CarbonGate Support

## Before opening a request

Start with the English [README](README.md), Chinese [README-CN](README-CN.md),
and the documentation in `docs/`. Include the CarbonGate version, operating
system, host integration, and the smallest safe reproduction when asking for
help.

## Where to ask

- Use a GitHub Discussion for usage questions, configuration help, and ideas
  that are not yet ready to become a feature request.
- Use the bug-report template for reproducible defects.
- Use the feature-request template for a clearly scoped enhancement.

Do not include API keys, tokens, personal data, customer data, or production
logs in public requests. For a suspected vulnerability, follow
[SECURITY.md](SECURITY.md) and use private vulnerability reporting.
