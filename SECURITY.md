# Security Policy

CarbonGate is pre-release software and has not received an independent security
audit. Do not rely on the CLI launcher as an OS sandbox.

Please do not open public issues containing exploit details or real credentials.
Use [GitHub private vulnerability reporting](https://github.com/soya7700/CarbonGate/security/advisories/new).
Include the affected version, operating system, entry point, safe reproduction
steps, and the security impact. Do not attach production logs or customer data.
